About these datasets & columns

All columns reflect raw NHL data, with the exception of teamAbbrev, which is added during the publish process for consistency across datasets. For example, in all team datasets, teamAbbrev is derived by mapping teamFullName using a reference lookup from Skater Bios.

This standardized field ensures consistent team identification across datasets and supports filtering, grouping, and downstream analysis workflows.

These datasets can support historical analysis, dashboards, and custom predictive modeling workflows.

Source data may be revised or updated over time.
